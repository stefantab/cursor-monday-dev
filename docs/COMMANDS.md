# Commands

- Update monday item: Opens selector with recent items inferred from branch.
- Start progress / Blocked: One-tap status changes with optional note.
- Add AI summary: Generates a concise, factual update from last N commits/PR.
- Link branch/PR: Maps current branch or PR to a monday item.

Tips:
- Use branch names like DEV-123/feature-title to improve suggestions.
- All AI summaries require manual approval before posting.
